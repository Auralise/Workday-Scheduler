# Workday Scheduler
Bootcamp Week 5 - jQuery and Bootstrap powered Day planner

## Purpose
This is a workday scheduler program designed to allow you to create items that are saved for each hour period of the working day (9am-5pm). 

## Built with
- HTML
- CSS
- jQuery
- Bootstrap CSS
